프로젝트 개요

본 프로젝트는 “Predicting Coupon Redemption” 데이터세트를 활용하여 할인 쿠폰 사용 패턴을 분석하고, 종합적인 분석결과를 Tableau 대시보드에 시각화하여 직관적인 인사이트를 도출하는 것입니다.

1) 주요 질문

  • 쿠폰 사용 패턴 분석: 어떤 조건하에서 고객들이 할인 프로모션에 더 민감하게 반응하는가?

    : 연령대, 성별, 가족 구성원 수 등 다양한 변수를 통해 고객이 어떤 조건에서 쿠폰을 더 자주 사용하는지 분석

  • 쿠폰 사용 후 구매 패턴 변화 분석: 할인 쿠폰 프로모션 후 할인 쿠폰을 사용이 고객의 구매 행동(금액, 빈도 등)에 어떤 변화를 가져오는가?

    : 고객들이 할인 쿠폰을 사용한 후 고객별 매출의 변화, 구매 빈도가 어떻게 달라지는 지 분석

2) 사용 데이터 소개

Predicting Coupon Redemption (Kaggle)

: 이 데이터세트는 다양한 할인 쿠폰 캠페인에 대해 고객들이 쿠폰을 사용했는지 여부와 관련된 정보를 포함

테이블설명
Train각 고객이 받은 쿠폰과 해당 쿠폰이 사용되었는지 여부를 포함하는 테이블
Campaign Data각 캠페인에 대한 정보 (캠페인 시작일/종료일, 캠페인 유형)
Coupon Item Mapping각 쿠폰이 어떤 상품에 대해 할인되는지에 대한 매핑정보
Customer Demographics고객의 인구 통계학적 정보 (나이, 결혼 여부, 소득 수준 포함)
Customer Transaction Data각 고객의 거래 정보 (구매 날짜, 구매 품목, 구매 금액 등)
Item Data판매되는 상품에 대한 정보 (상품 카테고리, 브랜드 등)

분석 결과

1) 쿠폰 사용 패턴 분석

분석 목표

: 연령대, 결혼 여부, 가족 구성원 수 등 다양한 변수를 통해 고객이 어떤 조건에서 쿠폰을 더 자주 사용하는지 확인

분석 방법

: 다양한 조건(연령대, 결혼 여부, 가족 구성원 수)에 따른 쿠폰 사용 비율을 시각화하고, 통계적으로 검정

분석 결과

  • 전체 거래 중 할인 쿠폰을 사용한 거래 비율

　　- 분석 기간 동안 전체 거래 중 할인 쿠폰을 사용한 거래 비율은 약 0.9%

  • 연령대별 쿠폰 사용

　1) 시각화

　　- 25세 이전의 어린 세대의 경우 다른 세대에 비해 할인 프로모션에 크게 민감하게 반응하지 않음

　　- 중장년층(46-55세) 세대의 쿠폰 사용 비율이 가장 도드라짐

　2) 카이제곱 검정 결과

H0(귀무가설): 연령대별 쿠폰 사용 패턴에 차이가 없다.

H1(대립가설): 연령대별 쿠폰 사용 패턴에 차이가 있다.

StatisticValue
Chi2 통계량16.269269
p-value0.006116

: 카이제곱 검정 결과 귀무가설을 기각하여 연령대별로 쿠폰 사용 패턴에 유의미한 차이가 있음.

  • 결혼 상태별 쿠폰 사용

　1) 시각화

　　- 기혼 고객이 미혼 고객에 비해 쿠폰 사용 비율이 더 높게 나타남

　2) 카이제곱 검정 결과

StatisticValue
Chi2 통계량3.019551
p-value0.082266

: 시각적으로는 기혼과 미혼의 쿠폰 사용 비율에 차이가 있는 것 처럼 보이나, 카이제곱 검정 결과 귀무가설을 기각하지 못하여 결혼 여부별로 쿠폰 사용 패턴에 유의미한 차이가 없다고 볼 수 있음

  • 가족 구성원 수별 쿠폰 사용

　1) 시각화

　　- 가족 크기가 1~3인 가구의 경우에는 쿠폰 사용 비율이 유의미하게 높게 나타나지 않음

　　- 4인 이상 가구의 경우 쿠폰 사용 비율이 높게 나타남

　2) 카이제곱 검정 결과

H0(귀무가설): 가족 구성원 수에 따른 쿠폰 사용 패턴에 차이가 없다.

H1(대립가설): 가족 구성원 수에 따른 쿠폰 사용 패턴에 차이가 있다.

StatisticValue
Chi2 통계량16.884339
p-value0.002036

: 카이제곱 검정 결과 귀무가설을 기각하여 가족 구성원 수 별로 쿠폰 사용 패턴에 유의미한 차이가 있음.

　　- 가족 구성원 수가 4인 이상인 경우 할인 프로모션에 민감하게 반응

2) 쿠폰 사용 후 구매 패턴 변화 분석

분석 목표

: 할인 쿠폰 사용 전후 고객의 구매 패턴 변화를 분석하여 할인 프로모션이 고객의 구매 금액과 빈도를 증가시켜 어떻게 매출에 기여하는지 평가

분석 방법

  • 분석 대상자

    : 프로모션 기간 동안 할인 쿠폰을 사용한 이력이 있으며, 첫 할인 쿠폰 사용 시점 전후 2개월간 구매 이력을 분석할 수 있는 고객

  • 분석 방법

    : 고객의 첫 할인 쿠폰 사용 시점을 기준으로 이전 2개월과 이후 2개월 동안 고객별 총 매출 금액과 구매 빈도를 비교하여, 전후 차이를 시각화 및 통계적으로 검정

분석 결과

  • 총 매출 금액 변화

　　- 첫 할인 쿠폰 사용 후 고객당 평균 매출액 약 34.5% 증가

  • 구매 빈도 변화

　　- 첫 할인 쿠폰 사용 후 고객당 평균 구매빈도 약 100회 → 130회로약 30% 증가

  • 통계 검정 결과(T-test)
가설 내용통계 검정 결과 (P-value)
구매 금액 변화<0.001***
구매 빈도 변화<0.001***

⇒ 첫 쿠폰 사용 이후 구매 금액 및 구매 빈도의 변화에 통계적으로 유의미한 변화가 있으며, 할인 쿠폰을 사용한 고객은 할인 쿠폰 사용 후 해당 플랫폼에서 더 많은 지출 및 구매 행위를 한다

결론 및 제언

1) 결론

쿠폰 사용 패턴 분석

  • 25세 이상 70세 미만, 특히 중장년층(46-55세)과 가족 구성원이 많은 고객은 할인 프로모션에 더 민감하게 반응
  • 결혼 여부는 쿠폰 사용 패턴에 큰 영향을 미치지 않음

쿠폰 사용 후 구매 행동 변화

  • 첫 쿠폰 사용 이후 총 매출 약 34.5%, 구매 빈도 약 30% 증가
  • 통계적으로 유의미한 변화 확인(p-value < 0.05)

2) 제언

  • 타겟 마케팅 강화
    • 어린 청년 보다는 25세 이상, 특히 중장년층 대상으로 연령대별 타겟 아이템에 대한 프로모션 진행
  • 장기적인 고객 분석
    • 첫 쿠폰 사용 후 매출 상승 효과가 지속되는지 장기적인 분석 진행
